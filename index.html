¿Qué es HTML?

HTML es como los ladrillos de una casa. Con él, construimos páginas web. ¡Imagina que eres un arquitecto del internet!.

¡Hola, mundo! en HTML

Este es el código HTML para crear tu primera página web:


<!DOCTYPE html>
<html>
<head>
    <title>Mi primera página</title>
</head>
<body>
    <p>¡Hola,   
    mundo!</p>
</body>
</html>

Escribe este código en la pestaña "HTML" y haz click en el botón "Compilar". ¡Verás tu primer "Hola, mundo!" en acción!
